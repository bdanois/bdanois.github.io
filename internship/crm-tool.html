Interactive CRM Tool

The purpose of our CRM Tool web application is to allow users to create, edit and delete lists. They can also populate the list with items, and have the ability to edit and delete these items. The image below displays the CRM Tool before and after the populated lists with items


The main feature that makes our web application interactive is the ability to drag and drop lists. By selecting the arrow icon at the top of the list, the user can drag and drop it to their desired location. The items in the list are also draggable.
