Interactive CRM Tool

The purpose of our CRM Tool web application is to allow users to create, edit and delete lists. They can also populate the list with items, and have the ability to edit and delete these items.

The main feature that makes our web application interactive is the ability to drag and drop lists. By selecting the arrow icon at the top of the list, the user can drag and drop it to their desired location. The items in the list are also draggable.

The work was divided between me and my fellow intern. We first designed the layout using HTML and CSS. I designed the main layout of the grid using CSS Grid while she designed the navigation header.

We then implemented the drag and drop feature through the use of a javascript library called Dragula. We developed JavaScript objects for the lists and items and then organized those into a dynamic array that changed as the location of each item changed.
